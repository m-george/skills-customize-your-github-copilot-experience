
# 📘 Assignment: Hangman Game Challenge

## 🎯 Objective

Build the classic word-guessing game using Python strings, loops, and user input. You'll create a game where players guess letters to reveal a hidden word before running out of attempts.

## 📝 Tasks

### 🛠️ Build the Hangman Game

#### Description

Create a Hangman game that implements word guessing mechanics. The game should randomly select a word from a predefined list, accept letter guesses from the player, and track game progress including remaining attempts and guessed letters.

#### Requirements

Completed program should:

- Randomly select words from a predefined list
- Accept letter guesses and show current progress (_ _ _ format)
- Track incorrect guesses remaining
- End when word is guessed or attempts exhausted
- Display win/lose messages
